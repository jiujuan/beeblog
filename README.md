antblog 蚂蚁博客

## antblog蚂蚁博客功能

博客antblog功能：

用户模块：

- 用户注册、登录、登出

文章模块：

- 文章列表展示（分页）
- 文章详情页
- 文章tag
- 文章分类

后台管理：

- 文章分类、归档（按时间线）

- 文章的增、删、改、查（markdown编辑器）
- 文章tag标签
- 文章点赞、收藏管理
- 文章的图片管理
- 文章评论管理

## 技术栈

### 后端技术

用Go 语言，结合 Gin + GORM + Zap + Viper +validator+go-jwt+go-redis 技术栈。

- MySQL
- Redis

### 前端技术

Vue3 + Tailwind CSS + shadcn/ui +Vite+TS
